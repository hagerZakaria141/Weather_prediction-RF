# Rain Prediction using Machine Learning

## Overview
Machine learning classification project to predict whether it will rain tomorrow using historical weather data.

## Objective
Build a model to classify rainfall occurrence (Yes/No).

## Dataset
Weather data including temperature, humidity, rainfall, wind, and pressure.

## Workflow
- Data preprocessing and handling missing values  
- Feature engineering and encoding categorical variables  
- Training Random Forest Classifier  
- Model evaluation using accuracy and classification report  

## Tech Stack
Python | Pandas | NumPy | Scikit-learn | Matplotlib  

## Results
The model predicts rainfall based on weather patterns with reasonable performance.
